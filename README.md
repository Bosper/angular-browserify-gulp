FRONTEND APP BOILERPLATE
========================

Installation of application boilerplate:
----------------------------------------

- Install RUBY ^2.0.0
- Install gems:

  gem install sass
  gem install bootstrap-sass
  gem install sass-rails

- Install GLOBALLY:

  npm i -G gulp-cli
  npm i -G bower
  npm i -G browserify

- Run commands:

  npm install (includes postinstall commands for bower etc.)
  gulp build
  gulp
